# Json Faker
my fake json server

## Requirements
- Node.js 6.x

## Install
`npm install`  

## Usage
`npm test` : run db.json to test  
`npm start <filename>`  
`npm start db.json` : static file  
`npm start example.js` : programmatic file  
`node index.js` : custom your server(optional)
